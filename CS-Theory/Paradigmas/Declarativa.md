# Programación Declarativa

A diferencia del paradigma imperativo (donde das instrucciones paso a paso), aquí se describe la lógica del problema sin especificar el flujo de control.

## Tipos de Programación Declarativa
* **Programación Lógica:** Basada en proposiciones y reglas de inferencia. Utiliza bases de conocimiento para resolver problemas mediante deducción.
* **Programación Funcional:** Basada en la evaluación de funciones matemáticas y evitando el cambio de estado (datos inmutables).

## Elementos Lógicos
Utiliza conectivos lógicos para evaluar proposiciones:
- **Conjunción $\land$:** Verdadero solo si ambos son verdaderos.
- **Disyunción $\lor$:** Falso solo si ambos son falsos.
- **Condicional $\rightarrow$:** Falso solo si el antecedente es verdadero y el consecuente falso.
