# Programación Orientada a Objetos (POO)

Este paradigma organiza el software como una colección de objetos que contienen tanto datos (**Atributos**) como comportamiento (**Métodos**).

## Conceptos Clave
* **Clase:** El "plano" o plantilla que define las características de un tipo de objeto.
* **Objeto:** Una instancia específica de una clase.
* **Herencia:** Capacidad de crear clases nuevas a partir de clases existentes, heredando sus atributos y métodos.
* **Polimorfismo:** Capacidad de que diferentes objetos respondan de forma distinta al mismo mensaje o método.
* **Encapsulamiento:** Ocultamiento de los detalles internos del objeto, exponiendo solo lo necesario.
