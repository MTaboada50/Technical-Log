# Paradigmas de Programación

Un paradigma de programación es un enfoque o estilo de programación de software. No se refiere a un lenguaje específico, sino a la metodología utilizada para resolver problemas.

## Clasificación Principal

1. **[Programación Estructurada](./Estructurada.md):** Basada en el control de flujo lineal y modularización.
2. **[Programación Orientada a Objetos (POO)](./POO.md):** Basada en la interacción de entidades llamadas "objetos".
3. **[Programación Declarativa](./Declarativa.md):** Enfocada en describir "qué" hace el problema (Lógica/Funcional).
4. **[Programación Paralela](./Paralela.md):** Enfocada en la ejecución simultánea para optimización de hardware.
