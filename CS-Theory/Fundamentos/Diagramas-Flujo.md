# Diagramas de Flujo de Datos (DFD)

Representación gráfica de la secuencia lógica de un algoritmo.

## Simbología Estándar
* **Óvalo:** Inicio / Fin del programa.
* **Paralelogramo:** Entrada / Salida de datos (Lectura/Escritura).
* **Rectángulo:** Proceso (Cálculos, asignaciones).
* **Rombo:** Decisión (Condicionales con salidas SI/NO).
* **Cilindro/Papel:** Salida por impresora o almacenamiento.
* **Círculo pequeño:** Conector entre partes del diagrama.
