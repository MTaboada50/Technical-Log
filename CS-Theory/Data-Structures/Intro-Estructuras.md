# Introducción a las Estructuras de Datos

Las estructuras de datos definen cómo se organiza la información para su procesamiento.

## 1. Estructuras Lineales
* **Arrays:** Almacenamiento contiguo. Ventaja: Acceso instantáneo. Desventaja: Insertar elementos es costoso.
* **Listas Enlazadas:** Cada elemento (nodo) conoce la ubicación del siguiente. Ventaja: Inserción dinámica.

## 2. Estructuras de Búsqueda Eficiente
* **Hash Tables:** Utilizan una función hash para distribuir datos en "buckets". Es la base de los diccionarios en Python.
* **Árboles Binarios:** Estructuras jerárquicas que facilitan la búsqueda y el ordenamiento.
